I designed the visual elements for a set of videos that were included in the release of GigJam. These videos included how to tips on the functionality of the app. I created mobile and desktop versions for each scenario.

In this project I had the opportunity to collaborate with the PM and motion teams. I worked with PM to come up with topics and then I created storyboards to represent the ideas visually. Then I was able to work closely with a couple of members of the motion team to hand off assets and revise the videos.
See all Videos

Storyboards

The first step in creating the design for this set of videos was to work with the PMs to determine which items needed additional explanation in the form of a how to video. Once we had this list, I created a storyboard for each video. These storyboards define the steps the user must take. I created rough sketches which I turned into high fidelity

Visual Assets

My team wanted to create a set of cohesive, but not identical videos, so I created visual elements unique to each video. These assets consisted of every item which moves individually in the animated video. The motion team used these assets to create the videos.
